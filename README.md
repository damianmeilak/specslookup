# SpecsRadar — Deploy to Vercel (Free)

## Deploy in 5 minutes — no coding needed

### Step 1 — Create a GitHub account
Go to https://github.com and sign up (free).

### Step 2 — Create a new repository
1. Click the "+" button → "New repository"
2. Name it: specslookup
3. Set it to Public
4. Click "Create repository"

### Step 3 — Upload these files to GitHub
1. On your new repository page, click "uploading an existing file"
2. Drag and drop ALL the files and folders from this zip:
   - /api/  (the whole folder)
   - /public/  (the whole folder)
   - vercel.json
   - package.json
3. Click "Commit changes"

### Step 4 — Deploy on Vercel
1. Go to https://vercel.com and sign up with your GitHub account
2. Click "Add New Project"
3. Find and select your "specslookup" repository
4. Click "Deploy" — no settings to change!

### Step 5 — Done!
Vercel gives you a live URL like:
   https://specslookup.vercel.app

Share it with anyone. It's free and stays live forever.
